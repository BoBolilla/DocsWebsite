# 检索式构建

1. 先在百度搜中文，看百科的英文名

2. 在知网学术词典搜中文，看出来的英文翻译，都囊括进去，这样就有多个表达方式了

   把同类英文单词尽可能搜集全面

3. 在wos高级搜索，例如下面是查找冠心病+外泌体的相关文献，其中外泌体只有一种说法exosome，冠心病有很多，推荐的做法是这样：

![img](./assets/a698d78e2127597a62d7211b7351bf404e09687c.png@690w_!web-note.webp)![image-20250928152143609](./assets/image-20250928152143609.png)多个词组成的话打引号

![image-20250928152511389](./assets/image-20250928152511389.png)

其中*通配符可以囊括单复数，而or用来包含冠心病的多种说法。

4. 数据库选择：用这两个就行了，中国科学引文包含了一些专利

![img](./assets/e2aec0e8be55d74c61e04d98dcfe2b70aecf3751.png@558w_!web-note.webp)

5. 先用标题搜一下

![img](./assets/a6398f9c9caed5824cc9a1b1bf16b02302130c6c.png@690w_!web-note.webp)

因为限定了标题，所以高亮都出现在标题，就可以看起来很方便，了解这两个词是怎么出现的，跟我的研究是否相关。

![img](./assets/9a17e94ad794f22fd4f3a178c78d19bc685e4146.png@690w_!web-note.webp)

这里搜索的时候可以看一下出来的条目数量，如果很少的话，就分开搜一下冠心病和外泌体，检查是否有表达错误。

这种检索方法也可以用来找研究思路，比如我自己本身做的是A方向，想跟B方向结合一下，就可以把两个合在一起搜，然后看看交叉点的地方别人都在做什么。

6. 然后用主题搜

![img](./assets/c7c1847d7ee5e1fd6a6dd8c98f834297c668746e.png@690w_!web-note.webp)

7. 日期降序

   如果降序侯前五条任然是跟自己研究方向相同，说明构建的检索式正确
