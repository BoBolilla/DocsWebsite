# 写作

## 局限性

1. 模型是否只针对某一种情况(针对威胁,只测试电击威胁;但是还有健康威胁,社交威胁)

2. •**图像质量**差异可能影响分割准确性。

   •横断面研究设计限制了**因果推断**。(是血管变化导致疾病哈还是疾病导致了血管变化)

   •研究聚焦于**特定年龄段和疾病类型**。

### 方法优化

1. 超参数设置, 是否找到最优组合
2. 不同数据集的任务设计和参数肯能存在差异,需要更多数据集验证特异性

### 样本与统计工作

1. 训练集样本量基于先前研究，但多变量神经影像模型的先验功效和样本量估算仍需发展(到底需要多少样本才能达到足够统计功效)。
2. 未来研究应考虑更大规模的样本以增强模型稳健性

## 未来方向

1. 在别的目标人群(患者等更有意义的)中进行测试验证

2. •进一步验证模型在**更多数据**上的泛化能力。

   •探索纵向数据以建立**因果关系**。

   •深入研究不同脑血管拓扑结构的变化。

   •开发基于这些发现的临**床辅助诊断工具**
