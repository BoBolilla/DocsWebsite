# **AI阅读工作流提示词**

1. 核心概述

“请仔细阅读这篇论文，提取出其核心内容，包括研究目的、主要方法、关键结论和创新点。用简洁明了的语言总结出论文的核心概述，字数控制在300字以内。重点突出论文的研究重点和贡献，避免过多细节描述，确保内容精炼且准确反映论文主旨。”

 

2. 翻译

将图片中的文字翻译成中文。

 

代码解读

“代码链接”+请从代码架构总览（markdown），关键技术实现（python），使用实例（python），这几个方面对以上链接代码进行解读。

 

3. 文章分析

请从“研究背景”，“研究方法”，“实验设计” “结果与分析” “总结结论与未来工作” 这五部分详细阐述这篇文献的内容。

 

什么是+”术语“+任务？

 

”术语“+是什么意思？作为超参，对提出的方法有什么影响？

 

4. 思维导图

请根据以上内容框架，用mermaid格式生成思维导图，要求清晰明了，水平方向生成。

 

5. 总结

在完成前面各部分的基础上，对整篇论文进行总结。总结内容包括论文的主要贡献、研究方法的创新点、实验结果的意义以及论文的不足之处和未来可能的研究方向。总结要全面且有条理，既要突出论文的优点，也要客观地指出其存在的问题，字数控制在500字左右，使读者能够通过总结快速了解论文的全貌和价值。
